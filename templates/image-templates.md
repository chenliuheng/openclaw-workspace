# 🎨 快速生图模板库

> 基于 awesome-gpt-image-2 整理的结构化提示词模板
> 用途：小红书配图、电商主图、信息图、封面等

---

## 📌 通用结构

```
[任务类型] + [主体] + [场景] + [风格] + [约束]
```

**核心约束**：
- 强制比例（9:16 或 1:1）
- 强制文字锁定（"文字必须清晰可读"）
- 强制主体放大

---

## 🧩 模板分类

### 1. 海报/封面（tpl-poster）

**常规模板**：
```
设计一张[活动/产品/电影]海报，主题为[主题词]。
主视觉：[主体元素]，标题文案：[标题]，副标题：[副标题]。
版式：[居中/左对齐/对角构图]，风格：[复古/未来/极简]。
色彩：[主色 + 辅色]，氛围：[情绪关键词]。
输出：9:16 竖版高分辨率海报。
```

**小红书爆款标题模板**：
```
生成一张 9:16 竖版小红书封面。
风格：Apple 发布会极简风，纯白背景，大量留白。
主标题：[大标题] 黑色加粗
副标题：[副标题] 灰色
底部：四个极简信息栏（图标+标题+短说明）
文字必须清晰可读，中文优先。
```

---

### 2. 信息图/对比图（tpl-infographic）

**常规模板**：
```
生成[主题]信息图，目标读者为[人群]。
结构：标题区 + [3-5]个模块（每模块含图标、短标题、1-2句说明）。
图表类型：[流程图/对比图/关系图/时间线]。
风格：[专业报告/科普插画]，主色[颜色]。
输出：信息层级清晰、可读性高的中文信息图。
```

**避坑指南**：
- 模块数量控制在 3-5 个
- 文字用短句，不要大段正文
- 避免画面混乱

---

### 3. UI/界面截图（tpl-ui）

**常规模板**：
```
为[产品类型]生成一张[平台，如 iOS/Android/Web]界面图。
核心功能：[功能点A]、[功能点B]、[功能点C]。
视觉风格：[极简/科技/拟物]，主色[颜色]，强调色[颜色]。
布局：[顶部导航/双栏/卡片流]，信息层级清晰，留白充足。
输出：高保真UI截图，文字清晰可读，9:16 比例。
```

**避坑指南**：
- 必须写明"平台 + 比例 + 布局"
- 强制要求"文字绝对可读"

---

### 4. 电商/商品（tpl-ecommerce）

**常规模板**：
```
生成[商品名]电商主图，卖点为[卖点1]、[卖点2]。
场景：[纯色棚拍/生活方式场景]，镜头：[特写/半身/全景]。
材质细节：[材质关键词]，灯光：[柔光/侧光/轮廓光]。
附加元素：[价格角标/卖点icon/促销文案]。
输出：电商平台可直接使用的商品展示图。
```

**避坑指南**：
- 材质和光影是灵魂
- 促销文案只给核心 1-2 句

---

### 5. 摄影/写实（tpl-photo）

**常规模板**：
```
拍摄主题：[人物/物品/街景]，场景为[地点]。
摄影参数风格：[35mm/85mm]，[浅景深/深景深]，[纪实/电影感]。
光线：[自然光/夜景霓虹/逆光]，情绪：[情绪词]。
细节要求：[肤质/材质/颗粒感]。
输出：高写实摄影风格图像。
```

**避坑指南**：
- 加入"皮肤纹理"、"胶片颗粒"增加真实感
- 用参数说话：`f/1.4` 代替"浅景深"

---

### 6. 插画/艺术（tpl-illustration）

**常规模板**：
```
创作[题材]插画，主角为[角色/主体]。
画风：[日漫/水彩/扁平/厚涂]，线条：[细腻/粗犷]。
配色：[配色方案]，背景：[简洁/复杂场景]。
构图：[近景/中景/远景]，重点表现[细节]。
输出：高质量插画。
```

**避坑指南**：
- 锁定笔触（如"厚涂"、"水彩晕染"）
- 慎用大师名，提取特征而非直接写名字

---

### 7. 品牌/标志（tpl-brand）

**常规模板**：
```
为[品牌名]设计品牌视觉方案。
品牌关键词：[关键词1]、[关键词2]、[关键词3]。
包含：Logo方向[几何/字标/图形]、辅助图形、主辅色。
风格：[现代/高级/亲和]，行业：[行业]。
输出：统一风格的品牌识别视觉图。
```

**避坑指南**：
- 必须强调"纯白背景"方便抠图

## ⚠️ 通用避坑清单

1. **比例必须写**：9:16 竖版 / 1:1 方形 / 16:9 横版
2. **文字必须锁**：写"文字必须清晰可读"
3. **禁止模糊指令**：不要写"做一张好看的图"
4. **文案硬编码**：标题、副标题都写死
5. **主体放大**：让主体占据画面 50%+ 视觉面积
6. **光线材质**：加"轮廓光"、"磨砂质感"等关键词
